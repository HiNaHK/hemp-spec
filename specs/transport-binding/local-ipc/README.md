# local-ipc

このディレクトリには、Local Process IPC Binding に関する未確定または将来整理対象の文書を置く。

このディレクトリ配下の文書は、現時点では v1.0.0 の確定済み Transport Binding specification 本文としては扱わない。

## 文書

- `specification.md`: Local Process IPC Binding の draft / working document。

今後、Local Process IPC Binding を正式な Transport Binding specification として扱う場合は、規範仕様として残す最小部分と、process / stdio / endpoint / realization category などの実現ガイド部分を分離して整理する。
