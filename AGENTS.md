# AGENTS.md

This repository contains HEMP specification documents.

## Repository roles

- `specs/` contains repository-ready HEMP specification documents.
- `validation/` is reserved for non-normative validation materials.
- `ROADMAP.md` is an implementation roadmap, not a protocol specification.

## Working rules

- Do not change HEMP protocol semantics unless explicitly instructed.
- Do not remove, ignore, or silently drop accepted HEMP premise documents.
- When reorganizing specification text, preserve the meaning of the accepted HEMP documents.
