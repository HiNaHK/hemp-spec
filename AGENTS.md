# AGENTS.md

This repository contains HEMP specification documents.

## Repository roles

- `specs/` contains HEMP specification documents.
- `validation/` contains non-normative validation materials.
- `ROADMAP.md` is an implementation roadmap, not a protocol specification.

## Working rules

- Do not change HEMP protocol semantics unless explicitly instructed.
- When reorganizing specification text, preserve the meaning of the HEMP specification documents.
- Treat `specs/` as the normative source for HEMP specifications.
